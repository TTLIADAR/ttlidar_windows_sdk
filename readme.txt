1. Download the sdk source.
https://github.com/TTLIDAR/ttlidar_windows_sdk

2. Open the sdk solution with Visual Studio.
ttlidar_windows\workspaces\sdk_and_demo\sdk_and_demo.sln

3.The sdk solution consists of two projects: a demo project and a lidar driver project 

4.You can run demo projects in Visual Studio that depend on lidar drive projects.

5.The operation result of the demo project is to print lidar information in the terminal

For details, refer to the user manual
